# Computer-Vision-Final-Project

This application can apply various filters on greyscale and RGB images

# Filters:
1) Low pass
2) High pass
3) Median
4) Gaussian
5) Sobel
6) Canny edge detector

# Other Features:
1) Histogram equalization
2) Hough circle Transform
